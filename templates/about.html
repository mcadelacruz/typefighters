about typefighters

typefighters is a multi-mode typing game focused on speed, accuracy, rhythm, and adaptability.

to play, enter your name, choose a mode, then type start to begin.

the game captures typing telemetry (dwell, flight, errors, pauses, speed) to be used for analytics and machine learning experiments.


developed by dela cruz, charles arvin p. for coe 005a - advance machine learning and deep learning
