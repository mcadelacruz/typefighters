about type it paster

type it paster is a typing game. type words as fast and accurately as you can.


developed by dela cruz, charles arvin p. for coe 003 - computational thinking with python

to play, type your name and then type start to begin. type the word shown below as quickly as possible. if you make a mistake, the game ends immediately. try to get the highest score you can!
