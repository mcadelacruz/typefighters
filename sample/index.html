Green Flames: Scented Candles

Completed: June 2023 A Capstone Scientific Research

This is a multi-part research project that I led called Green Flames: Scented Candle. It’s a sustainable project where we reused orange peels and used cooking oil to make eco-friendly scented candles. The idea was to turn common kitchen waste into something useful and relaxing. These candles not only help the environment by reducing waste but also support mental wellness through their calming scents.

Research Document

Here's the aftermath of this research project after our exposition.


 
 

 
 

Train City

Completed: June 2023 A Physics Project

This project was a team effort by our whole section, STEM204, led by me and three others. Our goal was to create a working train using real mechanical parts and recyclable materials, and to place it in a model of a sustainable city. We wanted to show how transportation and eco-friendly living can go together. Throughout the project, we faced a lot of challenges, but it helped us learn how to solve problems, work together, and apply what we've learned in class to something real and meaningful.

Type it Paster

Completed: December 2024 A game project for Data Structures and Algorithms

Type it Paster is a text-based game developed by me as a game project. The goal of the game is to type as many difficult long words under 2 minutes and get higher scores. It has a functional high scoring system stored in a text file and a bootleg timer system for the 2-min requirement.

Repository

 
 

 
 

Project Sunshine

Completed: May 2024 An NSTP Community Outreach Program

Project Sunshine is a group project for NSTP 2 where we go to different places to help and teach children. For this project, our group visited a nursery school where we spent time teaching the kids, giving them snacks, and organizing fun activities for them to enjoy. It was a meaningful experience that allowed us to share knowledge and bring joy to young children.

Documentation

 
 

 
 

Tavernhold

Completed: May 2025 A database project for Database Management System

Tavernhold is a medieval-themed online forum platform designed for aspiring mercenaries and adventurers. Built using Flask and SQLAlchemy, it features user authentication, threaded discussions, tagging, commendations (likes), notifications, and an experience-based leveling system. This forum site is designed with a handmade-styled UI with interactive sound effects and a tavern-inspired layout to enhance user immersion.

how this page was made...
 

When I started building this website, I wasn’t sure how to design it in a way that truly felt like my own.
While searching for inspiration, I discovered devportfolios.dev, a collection of developer portfolios.
Browsing through the list, I found areoladaniel.com, whose minimalist approach really resonated with me.
Inspired by that clean style, I set out to create a portfolio that reflects my personality—simple, modern, and uniquely mine.
